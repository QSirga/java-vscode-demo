## How to Run
1. Open this folder in **VS Code**
2. Make sure you have a **JDK installed** and Java set up in VS Code
3. Run:
   - Using VS Code: open `Main.java` → click **Run**
   - Or using terminal:
     - `javac Main.java`
     - `java Main`
